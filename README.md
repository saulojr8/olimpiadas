## **README.md**

### **Encontrando os craques do esporte! **

**O que faz essa aplicação?**

Imagina um buscador super rápido e fácil de usar, só que focado nos maiores atletas do Brasil! Com essa ferramenta,
você digita o nome de um jogador, uma modalidade esportiva ou qualquer
outra palavra-chave e pronto! A aplicação vasculha nossa base de dados e te mostra todos os
resultados relevantes, com nome, descrição e até um link para saber mais sobre o atleta.
É como ter uma enciclopédia esportiva na palma da sua mão!

**Como funciona a mágica?**

* **HTML:** Cria a estrutura da página, com campos para pesquisa, exibição dos resultados e um design bem clean.
* **CSS:** Deixa tudo bonitinho e organizado, com cores e fontes que combinam e uma interface fácil de usar.
* **JavaScript:** É o cérebro da operação! Ele pega o que você digita no campo de pesquisa, compara com nossa base
de dados e mostra os resultados na tela. Tudo isso de forma bem dinâmica, sem precisar recarregar a página.

**O que tem na base de dados?**

Nossa base de dados é um arquivo JavaScript chamado `dados.js`. Nele, temos um array com informações sobre diversos atletas,
como nome, descrição, tags e um link para uma página com mais detalhes. A cada nova pesquisa, o JavaScript varre esse
array inteiro pra encontrar os resultados que combinam com o que você digitou.

**Tecnologias utilizadas:**

* **HTML5:** Estrutura da página.
* **CSS3:** Estilização da página.
* **JavaScript:** Lógica da aplicação.

**Como usar:**

1. **Clique no link ao lado, na seção about, então você acessará o site. ** 
2. **Faça sua pesquisa:** Digite o nome do atleta ou esporte que você procura no campo de pesquisa e clique no botão "Pesquisar".

**Observações:**

* **Base de dados:** A base de dados atual pode ser encontrada no arquivo `dados.js`.
* **Melhorias futuras:** Planejo adicionar mais funcionalidades, como filtros por esporte, país e período.

**Divirta-se explorando o mundo dos esportes!** ⚽
